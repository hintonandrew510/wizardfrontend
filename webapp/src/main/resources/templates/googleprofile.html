The Google Slides integration lets use to dynamically create SCR Google Slides. In order to use Google Slides we must setup the turn on the Google account Slides account. Use the below steps to create a Google Profile
  1. Turn on the Google Slides API
  2. Create a Google Account
  3. Logon to your Google Account
  4. Select a project option
  5. Select a project creation option
  6. When you create your application, you register it using the Google API Console. Google then provides information you'll need later, such as a client ID and a client secret.
  7. Activate the Google Slides API in the Google API Console.
