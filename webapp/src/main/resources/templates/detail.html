Renaming the Wizard/SCR Name

Renamng the existing Wizard/SCR with the new name for the wizard/SCR. Remember that each Wizard/SCR should have a unique and meaningful name. The name distinguishes the wizards that you create.

Copying the Wizard/SCR

Copying the existing Wizard/SCR will copy all the data from the Wizard/SCR.

Deleting the Wizard/SCR

Deleting the existing Wizard/SCR will remove the Wizard/SCR from the system. You will not be able to recover the Wizard/SCR
